PacMan EXerCise

The PacMen Excercise has been one of the most fun activities I've done so far, it brings back memories from my childhood when I used to play videogames with the Atari system.

The code sets up a PacMen factory with two buttons used to create a PacMan and to start the game with loops and updates functions in order for PacMen to move randomly and in different directions each around the inside limits of the page as set within the script.

MTBA Route 1 Bus Locations Tracking in Real Time

This brings back some memories as to why I started learning about web apps and coding. I've always wondered how trucks can be tracked in maps on screens from an office and see their real time location many miles away. This has been a very satisfying exercise because know I know that any moving object that has a connection to the web can be tracked at different intervals.

The code is made of several async functions that fetch data from public APIs and then just simply runs functions to show current bus locations. I've added some colored icons in order to see the different buses with different directions as they are tracked in real time.
